# Templates Directory

This directory contains template files for common project documentation, including:

- Community guidelines
- Contributor documentation
- GitHub configuration files
- Issue templates
- PR templates

When implementing tasks from WORK_STREAM_TASKS.md that involve creating documentation,
refer to these templates as starting points, customizing them for your project's specific needs.

## Future Templates

Additional templates will be added here as they are developed, including:

- Code of Conduct templates
- Security policy templates
- Contribution guideline templates
- Issue templates (for both GitHub and repository-based approaches)
- PR templates
