# Z_Utils Source Code

This directory contains the source code for the z_Utils library, including examples and utility scripts.

## Directory Structure

- **/*.sh** - Main source scripts and examples
- **tests/** - Test scripts for z_Utils functions

## Key Files

### Templates and Examples
- **snippet_template.sh** - Template for Zsh snippet scripts
- **z_frame.sh** - Example of a full framework script
- **z_min_frame.sh** - Example of a minimal framework script

### Utility Scripts
- **audit_inception_commit-POC.sh** - Contains the most current version of z_Output() and other utilities
- **create_inception_commit.sh** - Script for creating inception commits
- **get_repo_did.sh** - Script for getting repository DIDs
- **z_output_demo.sh** - Contains test functions for z_Output

## Development Status

These scripts are in various stages of development and standardization. Refer to the `requirements/source_materials_inventory.md` file for detailed status information on each script.

## Coding Standards

All scripts in this directory should follow the coding standards defined in:
- `requirements/REQUIREMENTS-Zsh_Core_Scripting_Best_Practices.md`
- `requirements/REQUIREMENTS-Zsh_Snippet_Script_Best_Practices.md` (for smaller scripts)
- `requirements/REQUIREMENTS-Zsh_Framework_Scripting_Best_Practices.md` (for larger framework scripts)
